

Win ohjeet Request moduulin lataamiseen ja asennukseen.

1. Avaa cmd admin oikeuksilla.
    -Kirjoita windowsin hakuun cmd ja sitten oikea klikkaa ja paina run as administrator.
2. Sitten kun cmd (pääte) on auki kirjoita seuraava komento.
3. C:\Users\Your Name\AppData\Local\Programs\Python\Python36-32\Scripts>pip install requests
4. Vaihda "Your name" vastaamaan windows käyttäjääsi
Huom! Riippuen minkä python version olet asentanut nimi "Python36-32" voi olla jokin muu. Esimerkiksi Python312
5.Paina enter ja paketti latautuu sekä asentuu
6. Tämä komento myös olettaa että python on asennettu default sijaintiin.


Tässä linkki verkossa olevaan ohjeeseen: https://www.w3schools.com/python/module_requests.asp
